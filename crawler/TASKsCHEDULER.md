### 设置每日多次运行爬虫任务的详细步骤

#### 第1步：打开任务计划程序

1. 按 `Win` 键，或点击“开始”菜单。
2. 输入 “__Task Scheduler__” 或 “__任务计划程序__”。
3. 在搜索结果中点击并打开它。

#### 第2步：创建基本任务

1. 在右侧的 “__操作 (Actions)__” 面板中，点击 “__创建基本任务... (Create Basic Task...)__”。
2. __名称 (Name)__: 输入一个描述性的名称，例如 `Sydney Rental Crawler`。
3. __描述 (Description)__: （可选）输入更详细的描述，例如 `每天在11点、15点和18点运行悉尼租房爬虫`。
4. 点击 “__下一步 (Next)__”。

#### 第3步：设置第一个触发器

1. 在 “__任务何时开始？ (When do you want the task to start?)__” 界面，选择 “__每天 (Daily)__”。
2. 点击 “__下一步 (Next)__”。
3. 设置第一个运行时间。将开始时间设置为 __`11:00:00`__。
4. 确保 “__每隔 (Recur every)__” 设置为 `1` 天。
5. 点击 “__下一步 (Next)__”。

#### 第4步：配置要执行的操作

这是最关键的一步，您需要告诉任务计划程序运行哪个程序。

1. 选择 “__启动程序 (Start a program)__”。
2. 点击 “__下一步 (Next)__”。
3. 在 “__程序/脚本 (Program/script)__” 输入框中，您需要填入Python解释器&#x7684;__&#x7EDD;对路径__。

   - __如何找到Python路径？

     C:\Python313\python.exe__

     - 打开命令提示符 (cmd)。
     - 输入 `where python` 并按回车。
     - 它会显示类似 `C:\Users\YourUser\AppData\Local\Programs\Python\Python39\python.exe` 的路径。请复制这个完整路径。
   - 将复制的路径粘贴到 “__程序/脚本__” 框中。
   - 
4. 在 “__添加参数(可选) (Add arguments (optional))__” 输入框中，填入您的Python脚本&#x7684;__&#x76F8;对路径__：

   ```javascript
   crawler\v5_furniture.py
   ```
5. 在 “__起始于(可选) (Start in (optional))__” 输入框中，填入您项目&#x7684;__&#x6839;目录__。__这一步非常重要__，因为它确保脚本能正确找到其他相关文件。

   ```javascript
   d:\WEB-sydney-rental-hub
   ```
6. 点击 “__下一步 (Next)__”。

#### 第5步：完成并打开高级属性

1. 在摘要页面，检查所有设置是否正确。
2. __勾&#x9009;__&#x5DE6;下角的 “__当单击‘完成’时，打开此任务的属性对话框 (Open the Properties dialog for this task when I click Finish)__”。
3. 点击 “__完成 (Finish)__”。

#### 第6步：添加另外两个触发器

现在，任务属性窗口会自动打开，我们来添加下午3点和下午6点的触发器。

1. 在属性窗口中，切换到 “__触发器 (Triggers)__” 选项卡。
2. 您会看到已经存在的上午11点的触发器。点击 “__新建... (New...)__”。
3. 像第3步一样，选择 “__每天 (Daily)__”，并将时间设置为 __`15:00:00`__。点击 “__确定 (OK)__”。
4. 再次点击 “__新建... (New...)__”。
5. 选择 “__每天 (Daily)__”，并将时间设置为 __`18:00:00`__。点击 “__确定 (OK)__”。
6. 现在您应该能在列表中看到三个触发器：11:00, 15:00, 18:00。

#### 第7步：检查设置并保存

1. 切换到 “__设置 (Settings)__” 选项卡，可以根据需要调整一些选项，例如：
   - `如果计划的启动时间已过，则尽快运行任务 (Run task as soon as possible after a scheduled start is missed)` - 如果电脑在那个时间点关机了，这个选项很有用。
2. 点击 “__确定 (OK)__” 保存所有更改。
