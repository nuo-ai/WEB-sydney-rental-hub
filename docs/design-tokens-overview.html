所有截图请放入 docs/image/tokens/<category>/<token_name_with_underscores>.png。上传后即可在表格中直接预览路径。

截图建议从真实页面/组件中截取，并在图上标注对应 Token 名称，方便业务同学理解。

后续扩展建议

  • 补齐所有 Token 的截图，并在表格里勾选完成状态。
  • Storybook 中新增 Tokens/Overview 页面，与该表格共享数据源。
  • 调研并接入第三方 Playground（如 Backlight、Storybook token 插件），用于交互式调试。
  • 编写 lint / codemod 脚本，防止旧 Token 再次出现。
