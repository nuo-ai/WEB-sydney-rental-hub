# 🚀 快速启动指南

## 立即开始开发

### 1. 安装HBuilderX（推荐）
下载地址：https://www.dcloud.io/hbuilderx.html

### 2. 安装依赖
```bash
cd uniapp-miniprogram
npm install
```

### 3. 启动开发服务
```bash
# 微信小程序开发模式
npm run dev:mp-weixin

# H5网页开发模式  
npm run dev:h5
```

### 4. 在HBuilderX中打开项目
1. 打开HBuilderX
2. 文件 → 打开目录 → 选择 `uniapp-miniprogram`
3. 运行 → 运行到小程序模拟器 → 微信开发者工具
