# 设计令牌可视化指南

这些示例展示了 Sydney Rental Hub 设计系统的核心 Design Token 在前端的实际表现，帮助团队理解每个令牌在 UI 中的用途。

---

## 色彩（Colors）

```jsx
const colorTokens = [
  { token: '--brand-primary', label: '品牌主色', hint: '按钮、主强调' },
  { token: '--brand-primary-hover', label: '品牌主色 Hover', hint: '悬浮态' },
  { token: '--color-bg-primary', label: '背景 - 主背景', hint: '卡片/面板主体' },
  { token: '--color-bg-secondary', label: '背景 - 辅助', hint: '浅灰背景' },
  { token: '--color-selected-bg', label: '选中背景', hint: '列表选中态' },
  { token: '--color-border-default', label: '边框默认', hint: '分隔线/边框' },
  { token: '--color-text-primary', label: '文本主色', hint: '正文' },
  { token: '--color-text-secondary', label: '文本次级', hint: '辅助信息' },
  { token: '--color-danger', label: '警示红', hint: '错误状态' },
  { token: '--color-success', label: '成功绿', hint: '成功提示' },
]
```

<div style={{display: 'grid', gap: '16px', gridTemplateColumns: 'repeat(auto-fill, minmax(180px, 1fr))'}}>
  {colorTokens.map(({ token, label, hint }) => (
    <div key={token} style={{border: '1px solid var(--color-border-default)', borderRadius: '8px', overflow: 'hidden', display: 'flex', flexDirection: 'column', minHeight: '120px'}}>
      <div style={{height: '64px', background: `var(${token})`}} />
      <div style={{padding: '12px'}}>
        <div style={{fontWeight: 600}}>{label}</div>
        <div style={{fontFamily: 'var(--font-family-en-sans)', fontSize: '12px', color: 'var(--color-text-secondary)'}}>{token}</div>
        <div style={{marginTop: '4px', fontSize: '12px', color: 'var(--color-text-secondary)'}}>{hint}</div>
      </div>
    </div>
  ))}
</div>

## 间距（Spacing）

```jsx
const spacingTokens = [

  { token: '--space-md', label: 'md (12px)' },
  { token: '--space-lg', label: 'lg (16px)' },

  { token: '--space-2xl', label: '2xl (24px)' },
  { token: '--space-3xl', label: '3xl (32px)' },
]
```

<div style={{display: 'flex', flexDirection: 'column', gap: '12px', padding: '8px 0'}}>
  {spacingTokens.map(({ token, label }) => (
    <div key={token} style={{display: 'flex', alignItems: 'center', gap: '16px'}}>

      </div>

  ))}
</div>

## 使用建议

- **禁止硬编码**：新增样式时请优先引用 Design Token（`var(--token-name)`），避免写死颜色或尺寸。
- **并列组合**：复杂组件应结合多个令牌（例如按钮使用颜色 + 间距 + 阴影令牌）。
- **一致性检查**：若需要新增 Token，请先在设计评审中确认语义和命名规范。

> 所有示例均实时引用 CSS 自定义属性，可直接复制调整，以保证实现与设计系统保持一致。
