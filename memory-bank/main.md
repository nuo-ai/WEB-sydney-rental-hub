# 项目主规则 (Project Constitution)

## 最高优先级规则 (Top Priority Rules)
- **规则 0**: 在执行任何操作之前，必须首先完整阅读并理解本文档 (`main.md`) 和 `context-management.md` 中的所有规则。
- **规则 1**: 所有直接面向您的沟通（包括回复、提问、计划、代码注释和文档）都**必须**使用中文。内部思考过程（`<thinking>` 标签内的内容）应优先使用英文，以确保技术分析的严谨性和精确性。
- **规则 2**: 在开始任何编码或修复任务前，必须首先阅读 `activeContext.md` 和 `progress.md` 来同步当前状态。
- **规则 3**: 所有重要步骤都必须先向用户提议并获得同意。
