# SRH Tokens Playground

本页用 Nextra + Next.js 演示“移动端-小程序优先”的色板与组件交互效果：
- 成功按钮：hover = `#00670F`，active = `#008013`（与默认同色）
- 链接/副标题：`#6699CC`（text.subtitle 指向 link.primary）
- 支持 Dark/Light/Sytem 主题切换

import ThemeSwitcher from '../components/ThemeSwitcher'
import TokenButton from '../components/TokenButton'
import LinkSubtitleDemo from '../components/LinkSubtitleDemo'

<div style={{display: 'flex', gap: 16, flexWrap: 'wrap', alignItems: 'flex-start'}}>
  <ThemeSwitcher />
  <TokenButton />
</div>

> 说明
> - Tokens 来源：tools/style-dictionary/src/tokens/colors.dark.json / colors.light.json（等价映射到本页的 tokens/dark.js / tokens/light.js）。
> - 仅使用你提供的色板与明确指定的颜色，无新增随意色。
